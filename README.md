## Before Start
1. npm install 을 통해서 의존성 모듈을 설치 합니다.
2. 최상위 폴더에 .env 파일을 생성해 주세요.
    * .env의 내용은 .env.example을 참고해주세요.
3. npm start 명령어를 실행하면 서버와 DB가 연결됩니다.
